# VisAPI

___
_是一个开发友好的Python3.9+的Web框架_
___

## ✨ 特性

VisAPI提供以下特性:

* 👀 可视化API节点
* 🔄 Websocket支持
* 🚀 [RSGI](https://github.com/emmett-framework/granian/blob/master/docs/spec/RSGI.md) 支持
* ⚡️ [ASGI](https://asgi.readthedocs.io) 支持
* 📚 简单易学
